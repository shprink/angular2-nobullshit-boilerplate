Contains:

* Webpack config
* Sass scaffolding
* Routing capability
* Http request demonstration
* Bootstrap 4 Import
* Components
    * App (app scaffolding)
    * Home (homepage)
    * Navbar (menu)
    * HttpModule (Http call)
    * About (Info about this project)

## Install

`npm install`

## Commands

### Web Server

`npm run devserver`

then open `http://localhost:8080/s`

### Dump the dev app

`npm run dumpdev`

### Dump the prod (minified) app

`npm run dumpdev`
